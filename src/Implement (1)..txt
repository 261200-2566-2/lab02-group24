overview methods:
	
turn_on(){ : เป็น void เงื่อนไข
	ถ้า poweredOn ไม่เปิิดจะทำการ println ออกมาว่า model(รุ่นใดๆ) ให้ เปิด
	set ให้ poweredOn = true
	แต่ หาก poweredOn เปิดอยู่แล้วจะทำการ println ออกมาว่า model(รุ่นใดๆ) เปิดอยู่แล้ว
}

turn_off(){ : เป็น void เงื่อนไข
	ถ้า poweredOn จะทำการ println ออกมาว่า model(รุ่นใดๆ) ให้ ปิด
	set ให้ poweredOn = false
	แต่ หาก poweredOn ปิดอยู่แล้วจะทำการ println ออกมาว่า model(รุ่นใดๆ) ปิดอยู่แล้ว
